## 🏗️ Kubernetes Homelab

I co-build a bare-metal Kubernetes platform with [Waldemar Kubica](https://gitlab.com/ebi-droid) at **[Homelab](https://gitlab.com/homelab-dev)**. The main purpose of this project is to have a place to test and learn new technologies in the DevOps field.

**Stack:** NixOS hypervisors → Talos Linux cluster → Cilium BGP → FluxCD / ArgoCD GitOps

**My contributions:**
- **Democratic-CSI storage** — ZFS-over-iSCSI from TrueNAS, driver configuration, sealed secrets, storage classes
- **OpenEBS LVM** — local storage setup, Talos kernel module patches
- **Application deployments** — Ente Photos, Linkding, Mattermost (operator + CloudNativePG databases)
- **Custom Helm charts** — diagnostic-app and reusable common chart library
- **ArgoCD** — app-of-apps definitions, Gateway API HTTPRoutes
